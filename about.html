About This Page

These webpages were created by Martin Palanca, a 4th Year BCS Computer Science student, as part of Assignment 1 for the CPSC436I (Introduction to Industry Skills) course at the University of British Columbia. The primary functionality of this site allows users to:

  • Add custom messages into a Message Log . Messages are ordered from top to bottom from newest to oldest. Only the ten most recently entered messages will be viewable on the default view. All messages can be viewed using the "Expand" buttom at the bottom of the log.
  • Clear the existing Message Log. Users can also clear the existing Message Log using the "Clear Message Log" button.
  • Clear the input form. Users can clear existing input from the form field with the "Clear Form" button.
  • Start/Stop scrolling of messages. The default behaviour of the Message Log is to scroll messages from right to left. Message scrolling can be stopped with the "Stop Scroll" button, and resumed again with the "Start Scroll" button.
